Caropoly

This was a project we had to make in our first year at Howest. The assignment was to create our own theme of Monopoly. We got a little code from our professors, but we had to implement all the logic in the backend ourselves, and we had to decide on how we would make the "gameplay" in the frontend. So, in our group of 5 we decided to go with a car theme. I say group of 5 but in the end only 2 or 3 people actually did something. My contribution to the project was practically everything in both Java and JavaScript. I made about 75% of the backend and 70% of all the frontend JavaScript code. The styling in HTML and CSS was done by the others in my group. If you click the links below you can see the code for the backend and frontend. You can do basically everything in the game, except for trading and auctioning properties.
