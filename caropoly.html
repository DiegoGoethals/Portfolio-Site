Caropoly

This was a project we had to make in our first year at Howest. The assignment was to create our own theme of Monopoly. We got a little code from our professors, but we had to implement all the logic in the backend ourselves, and we had to decide on how we would make the "gameplay" in the frontend. So, in our group of 5 we decided to go with a car theme. I say group of 5 but in the end only 2 or 3 people actually did something. Because only 2 or 3 people actually did something, we had to do a lot of work in the end and the game isn't completely finished yet. I am talking about this project here to show you that I can work with Java as well as JavaScript, HTML and CSS. I am currently trying to get the rest of the bugs out of the game on my own and will post a link to the completed game when it's finished.
