Codenames

This was a project we had to make in our first year at UGent. The assignment was to create a basic version of the board game codenames. We had to make a small backend and some frontend. The backend was written in Python, using cgi, and the frontend was written in HTML, CSS and JavaScript. The backend was used to store the game state and the frontend was used to display the game state and to send requests to the backend. This was a project to showcase all things we learned in our first year about Python and JavaScript.
